Ext.Net.ComponentDragger Class Reference

A subclass of Ext.dd.DragTracker which handles dragging any AbstractComponent. This is configured with a AbstractComponent to be made draggable, and a config object for the Ext.dd.DragTracker class. A delegate may be provided which may be either the element to use as the mousedown target or a Ext.DomQuery selector to activate multiple mousedown targets. More...

Detailed Description

A subclass of Ext.dd.DragTracker which handles dragging any AbstractComponent. This is configured with a AbstractComponent to be made draggable, and a config object for the Ext.dd.DragTracker class. A delegate may be provided which may be either the element to use as the mousedown target or a Ext.DomQuery selector to activate multiple mousedown targets.
