◆ RendererHandler

virtual string Ext.Net.MVC.ColumnBaseAttribute.RendererHandler
getset

A renderer is an 'interceptor' method which can be used transform data (value, appearance, etc.) before it is rendered.

See Ext.util.Format for some default formatting functions.

The render function is called with the following parameters: value : Object The data value for the cell. metadata : Object A collection of metadata about the current cell; can be used or modified by the renderer. Recognized properties are: tdCls, tdAttr, and style.

tdCls : String A CSS class name to add to the cell's TD element. tdAttr : String An HTML attribute definition string to apply to the data container element within the table cell (e.g. 'style="color:red;"'). style : String

record : Ext.data.record The record for the current row rowIndex : Number The index of the current row colIndex : Number The index of the current column store : Ext.data.Store The index of the current column view : Ext.grid.View The current view Returns: The HTML string to be rendered.
