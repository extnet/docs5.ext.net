virtual ComponentDirectEvent Ext.Net.PropertyGridDirectEvents.BeforePropertyChange
get

Fires before a property value changes. Handlers can return false to cancel the property change (this will internally call Ext.data.Model.reject on the property's record).

Parameters source : Object The source data object for the grid (corresponds to the same object passed in as the source config property). recordId : String The record's id in the data store value : Object The current edited property value oldValue : Object The original property value prior to editing
