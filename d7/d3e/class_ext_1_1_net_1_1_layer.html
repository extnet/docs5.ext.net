 Sets the z-index of this layer and adjusts any shadow and shim z-indexes. The layer z-index is automatically incremented depending upon the presence of a shim or a shadow in so that it always shows above those two associated elements. Detailed Description

An extended Ext.Element object that supports a shadow and shim, constrain to viewport and automatic maintaining of shadow/shim positions.

◆ SetZIndex()

virtual Layer Ext.Net.Layer.SetZIndex ( int  zindex)
inlinevirtual

Sets the z-index of this layer and adjusts any shadow and shim z-indexes. The layer z-index is automatically incremented depending upon the presence of a shim or a shadow in so that it always shows above those two associated elements. Any shim, will be assigned the passed z-index. A shadow will be assigned the next highet z-index, and the Layer's element will receive the highest z-index.
