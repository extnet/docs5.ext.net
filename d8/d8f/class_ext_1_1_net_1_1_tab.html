◆ Href

virtual string Ext.Net.Tab.Href
getset

The URL to open when the tab button is clicked. Specifying this config causes the tab button to be rendered with an anchor (An element) as its active element, referencing the specified URL.

This is better than specifying a click handler of function() { window.location = "http://www.sencha.com" } because the UI will provide meaningful hints to the user as to what to expect upon clicking the button, and will also allow the user to open in a new tab or window, bookmark or drag the URL, or directly save the URL stream to disk.

See also the hrefTarget config.
